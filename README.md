# Can Culture Media – Launch-Ready React Site

**What’s included**
- Full styled site (Tailwind + lightweight UI components + animations)
- Calendly embed (30-min link)
- Netlify Forms contact form (set notifications to rich@canculturemedia.com in Netlify)
- Resources wired to /public/assets/*.pdf (replace with your real PDFs)

## Quick Start
```bash
npm install
npm start
```

## Build & Deploy
```bash
npm run build
```
- Push to GitHub → Netlify: New site from Git → Build: `npm run build` → Publish dir: `build`.
- Add domain in Netlify → point GoDaddy DNS.

## Replace PDFs
- Put your real files at:
  - `/public/assets/capstone-one-pager.pdf`
  - `/public/assets/lpaas-executive-summary.pdf`
